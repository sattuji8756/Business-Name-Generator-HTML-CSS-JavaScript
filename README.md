# 🚀 Business Name Generator

A simple and interactive **Business Name Generator Web App** built using **HTML, CSS, and JavaScript**.
It generates creative business names with one click and allows users to copy the name or download it as a logo image using the Canvas API.

---

## ✨ Features

* 🎲 Random business name generation
* 🎨 Smooth text animation effect
* 📋 Copy to clipboard
* 🖼 Download as PNG logo
* 💻 Clean and responsive UI

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Canvas API

## 🎯 How It Works

* JavaScript randomly selects words from predefined arrays
* Displays the generated business name with animation
* Canvas API renders the name as an image
* Users can copy or download the generated logo
